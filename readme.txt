이 프로젝트는 내가 프로젝트하면서 필요로했던 기능들을 모을 것들이다.
개인적으로는 
 1. 나 자신의 학습 프로젝트이다. 여러가지 모듈들을 실험하고 테스트하는 공간이다. 
 2. 기억력이 나빠서 쓸때마다 구글링을 하게되는 소모적인 작업을 줄이고자하는 유틸리티성을 여러가지 만들어볼 예정이다.


여러가지 프로젝트를 만들어야할것 같아서 프로젝트를 포함하는 공간으로 만들어 보았다.
